# Atividade_POO
Atividade em sala de aula

Questão 1 - O que é um vetor em C++? Explique com suas palavras.

Questão 2 - Qual a diferença entre declarar um vetor com valor fixo e inicializá-lo com valores específicos?

Questão 3 - Explique o que significa índice e qual o intervalo válido para um vetor int notas[10];

Questão 4 - Crie um programa que:
  
  Possua um vetor de 5 números inteiros
    
  Peça ao usuário para digitar os valores
  
  Exiba todos os valores digitados

Questão 5 - Usando o vetor do exercício anterior, calcule e exiba:
    
A soma total dos números

A média aritmética dos valores

Questão 6 - Leia 8 números inteiros para dentro de um vetor e determine:

O maior valor e em qual posição ele está

O menor valor e em qual posição ele está

Questão 7 - Crie um vetor com 10 números digitados pelo usuário.

Depois peça um número X e informe:

Se X está no vetor

Em qual posição ele aparece pela primeira vez

Questão 8 - Crie um vetor de 6 nomes (strings).

O programa deve:
    
Ler os nomes
    
Perguntar um nome a ser buscado

Informar se ele existe no vetor
